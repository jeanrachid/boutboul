cours GIT

Git est un outil de versonnig et il permet d'avoir une traçabilité et un historique des projets

Il permet également de travailler en équipe: Envoyer nos modifications à nos collégues et récupérer les modifications de nos collégues

Dans la traçabilité des fichiers GIT nous donne toujours 3 infos: Qui? Quand? Pourquoi?

Commandes de bases

initialiser un projet:
git init
voir le statut des fichiers
git status
Ajouter des projets
git add
Enregistrer les fichiers ajoutés
git commit
un commit nécessite toujours un message
commit avec message:
git commit -m "message"
commit avec add + message:
git commit -a -m "message"
ajouter dans les fichiers:
git commit add .
corriger les modifications sur un fichier ( pas encore add )
git checkout 'fichier'
supprime un add:
git reset.
supprimer le dernier commit:
git reset HEAD^
supprimer les deux derniers commit:
git reset. HEAD^^

la notion de branche

git branch

une branche est une copie du projet, elle permet de developper en parallele de nouvelles fonctionnalités lorsqu'on considere les nouveautes operationnelles on peut les rappratier sur la versions principale (master)

commande pour voir les branches:
git branch
creer une branche:
git branch 'nom'
se placer sur une branche:
git checkout 'nom'
voir les differences entre 2 branches:
git diff 'nom'..master
